【開発アプリ】

IncidentAppというデスクトップアプリを作りました。
社内で使用している「顧客管理システム」が使いづらかった為、作ることにしました。
日々の業務活動の登録、検索、出力などが簡単に出来ます。
機能に関しては、学習と並行してアップグレードしていきます。


【開発環境】

○ C#
○ Visual Studio Community 2022
○ Microsoft SQL Server 2019
